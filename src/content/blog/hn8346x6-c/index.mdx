## 概述

HN8346X6-C是华为光猫的九洲代工低配版本，本文记录了破解、改华为界面和配置的探索过程。

## 设备信息

- **型号**：HN8346X6-C
- **版本**：V5r022
- **代工**：-C为九洲代工低配版

<Aside type="note">
8346x6-c的九州代工版本，芯片缩水，USB3.0没了，内存和flash小了一半，运营商界面不能改，只能restorehwmode使用，地区包能否补未知。

截至2025/1/16，无r22版本shell补全方法。
</Aside>

## Telnet连接

### 连接问题

**问题**：上不了telnet。

**解决方案**：
- 禁用猫内防火墙
- 禁用Windows防火墙

### 连接命令

```bash
telnet 192.168.1.1
```

### 默认账户密码

- **root** / **adminHW** 或 **Hw8@cMcc**

### Shell补全失败

```bash
su
shell
?
```

<Aside type="caution">
只有两条命令，未找到restorehwmode.sh与其他常用命令。
</Aside>

### 查看配置

```bash
exit
？
display current-configuration
```

**结果**：`password=""`，实测可telnet，但shell补全失败，无法看到密码。

## 配置恢复

### 忘记宽带账密

<Aside type="note">
不记得宽带固化账密，或者其他配置有误怎么办？

修改配置时断开光纤，需要修复时重置光猫并连接光纤，使用tr069自动恢复设置。
</Aside>

### 配置完成后

<Aside type="tip">
配置好后呢？

重置光猫并断开光纤，修改超管密码并删除tr069再连接光纤。
</Aside>

## 破解步骤

### 1. 准备工作

1. 拔光纤
2. 重启光猫
3. 打开华为ONT使能工具
4. 刷新网卡
5. 刷r020版本telnet shell

<Aside type="caution">
提示失败，但实际可能已经成功。
</Aside>

### 2. 重置和配置

1. 重置光猫
2. 输入超管账密：**CUAdmin**
3. 打电话问宽带密码
4. 删除tr069
5. 改桥接

<Aside type="note">
暂未测量不改华为界面是否桥接限速。
</Aside>

## 总结

HN8346X6-C光猫的破解和配置需要注意：

1. 九洲代工版本功能缩水
2. R022版本目前无法完全补全shell
3. 需要正确配置防火墙才能连接telnet
4. 配置前要断开光纤，配置后要删除tr069
5. 很多高级功能需要进一步探索
